# Bookingsystem 
#### Dette er en enkel nettside laget med Flask, HTML, CSS og Python for å gjøre det mulig å reservere fotosyudio og lydstudio. Nettsiden har en enkel registrerings og loginside koblet opp mot en database og med sider som gjør det mulig å reservere rom.

## Funksjoner: 
## Hjemmeside: en enkel login side
## Registreringsside : Gjør det mulig å lage en bruker for å logge seg inn

# Kjøre programmet lokalt
### Installere flask (dersom du ikke allerede har dette)

1. Sette opp ett virtual enviroment- python3 -m venv env
2. Aktivere miljøet- source env/bin/ activate
3. Installere flask- pip3 install flask flask_sqlalchemy
4. Navigere til prosjektmappen- cd sti/til/prosjekt
5. Åpne nettsiden i nettleser- 172.20.128.31
